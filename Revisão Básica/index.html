Comandos de estilização comuns do CSS:

  • Exemplo usando variáveis para usar cores específicas: --clearneonBlue: #4deeea

    Exemplo usando a variável clearneonBlue

  • Exemplo circulando a "caixa" (BoxModel) que esta sendo estilizada e deixando a borda curva: border: 1 solid var(--clearneonBlue); border-radius: 5px;

    Exemplo usando o border

  • Exemplo usando "padding" para preencher um espaço dentro da "box" (caixa) que esta sendo estilizada: padding: 10px;

    Exemplo usando o padding (usando uma borda para demonstrar melhor)

  • Exemplo usando a "margin" para alterar a distância de uma box para a outra: margin: 10px

    Exemplo usando a margin

  • Exemplo usando o "width" e "height para alterar o tamanho de uma box: width: 130px; height: 50px;"

    Exemplo com o tamanho da box alterada

  • Exemplo usando o "text align" para alinhar o texto e o "text decoration" para adicionar uma linha no texto: text-align: center; text-decoration: underline var(--deepneonRed);

    Exemplo alinhando o texto no centro e aplicando uma decoração

  • Exemplo usando o "text transform" para deixar o texto em caixa alta, usando o "letter spacing" para adicionar um espaço entre as letras e alterando o tamanho do texto de outro parágrafo: text-transform: uppercase; letter-spacing: 1px; font-size: 18px;

    Exemplo transformando o texto para caixa alta e adicionando um espaçamento nas letras

    Exemplo alterando o tamanho da fonte

  • Exemplo usando a "font style" para trocar o estilo de fonte, deixando outra fonte itálica e usando o "font weight" para alterar a grossura da fonte: font-family: fantasy; font-style: italic; font-weight: 1000;

    Exemplo com a fonte alterada

    Exemplo da fonte em itálico e mais grossa
